# premier_dbt
Mon premier projet avec DuckDbt

Ce dépôt contient divers exemples d'algorithmes et de modèles de données.

## Scripts Python inclus

- `nombres_premiers.py` : Recherche de nombres premiers en utilisant un crible d'Ératosthène segmenté.
- `markov_chain.py` : Simulation d'une chaîne de Markov (ex: modèle météo) avec calcul de la distribution stationnaire.
- `transportation_problem.py` : Résolution d'un problème d'optimisation du transport (minimisation des coûts) par programmation linéaire.

## Base de données (SQL)

- `schema.sql` : Structure d'une base de données de facturation simple.
- `donnees_exemple.sql` : Jeu de données pour tester le schéma de facturation.

## Installation

Pour exécuter les scripts Python, installez les dépendances nécessaires :

```bash
pip install numpy scipy rich
```
